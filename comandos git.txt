SUBIR UN PROYECTO

# 1. Cambiar a la rama dev
git checkout dev

# 2. Actualizar dev con lo último del remoto
git pull origin dev

# 3. Cambiar a tu rama de trabajo
git checkout feature/rodas

# 4. Hacer merge de dev en tu rama
git merge dev

# 5. Subir tu rama actualizada
git push origin feature/rodas
